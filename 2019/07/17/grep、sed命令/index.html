grep、sed命令

前言

昨天记录了一下awk的一般用法，有了awk，怎么能拉下其他两位兄弟呢🐶

三兄弟的主要区别：

  • grep适合单纯的查找或匹配文本
  • sed适合编辑匹配到的文本
  • 适合格式化文本，对文本进行比较复杂格式处理

####一、grep

Global search Regular Expression and Print out the line

顾名思义，grep是一个利用正则表达式进行全局搜索的工具，grep会在文本中按照指定的正则进行全局搜索，并将搜索出的行打印出来

当然，不使用正则也可以使用grep

写了这么多例子，还不如做个表格～🐍

命令 说明
–color 匹配文本着色
-i 忽略大小写
-n 显示结果所在行好
-c 匹配到的总行数（非匹配到的次数）
-o 只显示符合条件的字符串，非整行显示，每个符合条件的字符串单独显示一行
-v 输出不带关键字的行
-w 匹配整个单词
Ax，Bx 向前，向后输出
-e 逻辑or
-q 静默
-P 正则匹配

二、sed

轻量级流编辑器，主要用来将数据进行选取、替换、删除、新增的命令

命令 说明
-n 只输出sed命令处理的行
-e 允许对输入数据应用多条sed命令编辑
-i sed修改结果直接修改读取数据的文件，而不是屏幕输出
a 追加
c 替换
i 插入
d 删除
p 打印
s 字符替换 （格式：s/旧字符串/新字符串/g）
